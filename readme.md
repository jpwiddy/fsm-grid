# How to use?

$ npm i fsm-grid --save

# Integration

Should work out of the box with webpack, respectively angular-cli. All you need to do is to include `FsmGridModule`:

```ts
import { FsmGridModule } from 'fsm-grid';

@NgModule({
  imports: [FsmGridModule],
  ...
})
class AppModule {}
```
